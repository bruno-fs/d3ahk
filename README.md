# d3ahk
my ahk script used to autocast skills on d3

## Hotkeys

`Alt + 1`: start/stop spamming

`Alt + UP`: increase spam interval (default 300ms)

`Alt + DOWN`: decrease spam interval

You must choose which keys will be spammed

`Ctrl + Q`: spam Q

`Ctrl + W`: spam W

`Ctrl + E`: spam E

`Ctrl + R`: spam R
